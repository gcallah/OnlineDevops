1. Docker:The most popular tool for containerizing applications.
  2. Continuous Integration (CI):The process of integrating different developers' work with high regularity.
  3. Continuous Deployment (CD):Regularly deploying new work into production, usually with every push to the master branch on the origin source-code control server.
  4. Containerization:The process of packaging an application and everything it depends upon in a "container," then running it in its own virtual environment.
